# Code of Conduct

KAI OS follows the Contributor Covenant spirit: be respectful, constructive, and focused on making the project better.

## Expected Behavior

- Use welcoming language.
- Respect differing viewpoints.
- Give and receive technical feedback with care.
- Keep discussions focused on the project.

## Unacceptable Behavior

- Harassment or insults.
- Personal attacks.
- Publishing private information.
- Spam or abuse.

Maintainers may remove comments, issues, or pull requests that violate these expectations.
